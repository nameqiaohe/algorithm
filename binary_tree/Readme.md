# C 语言实现 二叉树

--------------------

## 1. BinaryTree insertBinaryTreeRecursive(BinaryTree tree, int val);
    (1)递归实现 二叉树的插入
    (2)插入规则：按照节点值的大小，左小、右大

## 2. BinaryTree insertBinaryTreeIteration(BinaryTree tree, int val);
    (1)迭代实现 二叉树的插入
    (2)插入规则：按照节点值的大小，左小、右大

## 3. void levelTraversal(BinaryTree tree);
    二叉树的层级遍历，借助队列来实现
